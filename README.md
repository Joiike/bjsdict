# bjsdict
bjsdict 是一款chrome翻译插件.<br>
1. 支持悬停取词, 双击取词.<br>
2. 支持汉英,英汉互译.<br>
3. 支持时间戳与日期格式字符串互转<br>
4. 支持快捷键
